This module will have a computer-like background. There are four buttons labeled song 1-4, a small screen under each diplaying a digit in hexadecimal accompanied by two arrows, and a button labeled "submit".

When clicked, each of the song buttons will play a short song, called a "sbemail song", that has a number associated with it. See here for a full list of sbemail song transcriptions. NOTE: In the case of more than one of the same number, such as 122a/122b or 193/193½ where the transcriptions are completely different, only the first of each can appear on the module.

Take the number associated with each sbemail song and convert it to binary, with zeros added to the left to make it 8 bits long if needed. Concatenate these to get a 32-bit string, and perform the operation associated with the background on each pair of bits (i.e. the 1st and 2nd bit, the 3rd and 4th bit, etc.; use the table above and to the left for reference). Arrange the resulting 16 bits into a 4x4 grid in reading order to get the initial grid.

On this module will be a stage display, a larger rectangular display below and a 4x4 grid of hexadecimal digits. Pressing the stage display will play a short song called a "sbemail song". Every time a non-ignored module is solved, this module will change which sbemail song is played.

See here for a full list of sbemail song transcriptions. NOTE: In the case of more than one of the same number, such as 122a/122b or 193/193½ where the transcriptions are completely different, only the first of each can appear on the module.

At each stage, take the number associated with each sbemail song and convert them to binary. Rearrange the binary number as shown in the table to the left depending on the name of the last solved module. On each pair of bits, perform the operation associated with the resulting number to make the sequence 4 bits long, then convert it to a hexadecimal digit. Modify the hexadecimal digit depending on conditions. After all modules are solved, submit the sequence of hexadecimal codes obtained.
